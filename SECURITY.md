# Security Policy

This is a **non-operational portfolio project**. Still, responsible disclosure is welcome.

## Reporting a vulnerability
If you discover a security issue, please open a **private** report (GitHub Security Advisories if enabled),
or contact the repository owner directly.

Please include:
- a clear description of the issue
- steps to reproduce
- potential impact
- suggested fix (if you have one)

Thanks for helping keep the project safe.
